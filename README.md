## How to run this project?

1. Create Virual Environment

```shell
conda create -n {env-name} python=3.10
```

2. Activate Viral Environment

```shell
source activate {env-name}
```

3. install package

```shell
conda install pypdf2 pandas
```

> **Deavtivate Virual Environment**
>
> ```shell
> source deactivate
> ```
